All skills live under `skills/engineering/`.

Every skill must have a reference in the top-level `README.md` and an entry in `.claude-plugin/plugin.json`.

Each skill entry in the top-level `README.md` must link the skill name to its `SKILL.md`.

The `skills/engineering/README.md` lists every skill with a one-line description, with the skill name linked to its `SKILL.md`.
